Livable Cities e-magazine
=========================

Open `index.html` in a browser to view the one-page multimodal e-magazine.

Project files:

- `index.html` contains the publication structure, article content, placeholders, references, and visual credits.
- `styles.css` contains the responsive e-magazine design system.
- `script.js` handles the mobile menu and active navigation state.
- `assets/` contains the four housing affordability visuals.

To add teammate content later, replace the bracketed placeholder titles, bylines, notes, visual placeholders, and reference placeholders in the Urban Sustainability and Public Transport sections of `index.html`.
